A simple CLI is included in `bml`. To easily access it, install `bml`
globally with:

```sh
npm install -g bml
```
It supports reading `bml` markup from stdin or a file and outputting it
to stdout. For more details, help is available from the CLI.

```sh
bml --help

# or

man bml
```
